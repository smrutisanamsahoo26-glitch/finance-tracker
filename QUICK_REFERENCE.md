## 🎮 How to Use

### Add a Transaction

1. Click "+ Add Transaction"
2. Fill in: Title, Amount, Category, Type
3. Select Date and add Notes (optional)
4. Check "Recurring" if needed
5. Click "Add Transaction" ✅

### View Transactions

1. Go to Transactions page
2. Search by title/notes
3. Use filters (category, type, date)
4. Sort by date/amount
5. Edit or delete as needed

### Set Budget

1. Go to Budget page
2. Click "Update Budget"
3. Enter monthly amount
4. Track spending against budget
5. View category breakdown

### Analyze Spending

1. Go to Analytics page
2. View 12-month trends
3. See category distribution
4. Check spending insights
5. Track recurring transactions

## 🆘 Quick Troubleshooting

**Issue**: Port 5173 in use

```bash
# Vite will auto-use next port or:
# Edit vite.config.js and change port
```

**Issue**: Styling not working

```bash
# Reinstall Tailwind
npm install tailwindcss postcss autoprefixer
npm run dev
```

**Issue**: Dependencies not installing

```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```
